What is bootstrap

Bootstrap provides faster, easier web development using a front-end framework. It is free. There are hundreds of HTML and CSS based design templates with Bootstrap, including typography, forms, buttons, tables, navigation, modals, image carousels, and more. There is also optional JavaScript plugins with built on jQuery. You can also build responsive websites with Bootstrap

Bootstrap Components

Bootstrap comes with lots of resourceful components. This component set allows you to create a complete static website with little effort. Grid, carousel, navbar, table, card, modal, form, etc. are some of the main components.

Boostrap Container

Among Bootstrap's layout elements, containers are the most basic. By using a container, you set the content's margins to accommodate responsive behavior in your layout. This contains the row elements, which in turn contain the columns (called the grid system). We can see two types of container in bootstrap.

Boostrap Carousel

Bootstrap carousels allow you to include a slider with indicators and controls on your website using a flexible, responsive method. This is a very responsive and versatile component that can accommodate any type of content you want to add, including images, iframes, videos, or any other type of media.

Boostrap Navbar

A navigation bar is similar to a navigation header situated near the top of the page. Depending on the screen size, it can be collapsed or extended. At the top of the page, create a standard navigation bar by adding <nav class="navbar navbar-default">

Boostrap Grids

Bootstrap's grid is built using Flexbox, which supports up to 12 column layouts across the page. In addition to using each column individually, you can group the columns together to make wider columns. As the screen size increases or decreases, the columns in the grid will re-arrange themselves automatically.

Boostrap Collapse

In order to hide and show content, bootstrap collapse is used. It is a built-in JavaScript plugin. The ".collapse" class is used to specify the collapsible element.

Boostrap Buttons

In Bootstrap, you can add a button in nine different styles. Each of them has a different look and a different semantic purpose. "btn-primary", "btn-success", "btn-info", "btn-danger" etc are the class name of bootstrap buttons.

Boostrap Jumbotron

Basically, a Jumbotron allows a web page or website to highlight the most important or most essential elements. It comes with a big grey box by default, allowing you to use any bootstrap components within.

Boostrap Dropdown

You can use dropdowns to display links and more as toggleable contextual overlays. A JavaScript dropdown plugin included in Bootstrap makes them interactive. They’re toggled by clicking, not by hovering. A basic dropdown menu is defined by the .dropdown class.

Boostrap Alerts

In Bootstrap Alerts, pre-defined alert messages are easily created with just a few clicks. Alert makes your messages more appealing to your users by adding a stylish appearance.

Boostrap Form

Bootstrap forms are widely used. It allows you to style All textual <input>, <textarea>, and <select> elements with class .form-control have a width of 100%.

Boostrap List Group

You can create a list group using bootstrap in order to create individual list items. A list group consists of items in an unordered list. A basic list group is created by using the classes ".list-group" and ".list-group-item" within the element.

Boostrap Modal

An extension of the bootstrap framework, the modal plugin displays a popup window on top of a current page. You can use this modal to display content, video, image, anything you want.

Boostrap Pagination

The purpose of pagination is to organize the pages of your website. If you have a lot of web pages on your website, it becomes essential.

Boostrap Tooltips

In Bootstrap 4, a tooltip is also known as an infotip or screentip. The graphical user interface (GUI) element serves to display information about items without requiring the user to interact with them .

Boostrap Popovers

Popovers in bootstrap work similarly to tooltips. When the user clicks on an element, it appears as a pop-up box. It has the advantage of containing more information than a tooltip.
